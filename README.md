# Banking Customer Analytics (SQL • Excel • Power BI • Tableau)

Project Overview  
End-to-end analytics project performed on 50,000+ customer records using SQL, Excel, Power BI, and Tableau.  
Includes transaction trends, loan analysis, product usage insights, and dashboards for financial KPIs.

 Tools Used  
- SQL  
- Excel  
- Power BI  
- Tableau  

Key Analysis  
- Credit & debit transaction summary  
- Loan amount and customer segmentation  
- Branch-wise performance  
- Product usage analysis  
- Monthly transaction trends  

 Dashboards Included  
- Power BI: KPI Overview, Deposit vs Loan, Branch Performance  
- Tableau: Customer Segments, Monthly Trends  

 Files Included  
- SQL Query Scripts  
- Power BI Dashboards  
- Tableau Workbooks  
- Reports & Visuals
